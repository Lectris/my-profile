# **仕様書 (Specification Document)**

## **1\. プロジェクト概要**

* **名称:** ポートフォリオ兼職務経歴書  
* **目的:** 自身のスキル（実務×エンジニアリング）を体現するポートフォリオとして機能しつつ、企業提出用の職務経歴書生成ツールとしての役割も果たすこと。  
* **ターゲット:** 採用担当者、企業のDX推進部門、知財管理部門。

## **2\. デザイン・UI/UX仕様**

### **2.1. Web表示モード (Screen)**

* **テーマ:**  
  * **Dark Mode (Default):** ブランドカラー（Navy: \#1e3a8a, Slate: \#0f172a）とアクセント（Gold: \#b45309）を使用した高級感のあるデザイン。  
  * **Light Mode:** 視認性を重視した白背景・濃紺テキストの清潔感あるデザイン。トグルボタンで切り替え可能。  
* **ナビゲーション:**  
  * **PC (lg以上):** 左側固定サイドバー。スクロール位置に応じてアクティブなメニューがハイライトされ、該当するアコーディオンメニューが自動展開する「ScrollSpy」機能を実装。  
  * **Mobile (md以下):**  
    * **Top:** スクロールするとセクション見出しが画面上部に吸着（Sticky）し、現在地を明示。  
    * **Bottom:** 固定ナビゲーションバーによる親指操作エリアの確保。  
* **アニメーション:**  
  * IntersectionObserver を使用し、要素が画面内に入ったタイミングでフェードイン・スライドアップ表示 (.reveal-on-scroll)。  
  * カード要素へのホバー時、Z軸方向への浮き上がりと発光表現。

### **2.2. 印刷表示モード (Print / PDF Export)**

ブラウザの印刷機能実行時に適用されるCSS (@media print) 仕様。

* **表示制御:** Web用ナビゲーション、背景装飾、トグルボタン、アニメーション要素をすべて非表示 (display: none)。  
* **レイアウト:** A4用紙サイズに最適化。背景色を白、文字色を黒に強制統一。  
* **ヘッダー挿入:** 印刷時のみ「職務経歴書」タイトル、氏名、現在日付、略歴要約を表示。  
* **QRコード:** 紙面からWeb版へ誘導するため、GitHub URLのQRコードとテキストリンクをヘッダー下部に自動生成。  
* **改ページ制御:** 職務経歴の各ブロック (.project-item) やチャートがページの途中で切れないよう break-inside: avoid を適用。

## **3\. 機能要件**

### **3.1. コンテンツ管理**

以下のセクションで構成される。

1. **Hero:** キャッチコピーと主要な価値提案。  
2. **About (自己PR・展望):** 強みの要約、コンピテンシー（レーダーチャート）、志望動機・今後の展望。  
3. **Experience (職務経歴):** 具体的な業務内容と実績（数値含む）。  
4. **Skills (スキルセット):** 技術スタックの棒グラフ、カテゴリ別スキル詳細、保有資格一覧。  
5. **Portfolio (開発実績):** 個人開発・業務開発ツールのGitHubリポジトリ紹介。

### **3.2. インタラクティブ機能**

* **排他的アコーディオン (PC):** サイドバーのメニューをクリックまたはスクロール連動で開閉。他グループが開くと現在グループ以外は閉じる仕様。  
* **データ可視化:**  
  * **Chart.js:** レーダーチャート（コンピテンシー）、横棒グラフ（技術スキル）。テーマ切り替え時に軸の色などを自動調整。

### **3.3. セキュリティ・保護機能**

* **検索除外:** \<meta name="robots" content="noindex, nofollow" /\> により検索エンジンのインデックスを拒否。  
* **コピーガード (抑止力):**  
  * CSS: user-select: none でテキスト選択を視覚的に無効化。  
  * JS: contextmenu (右クリック), copy, cut, selectstart イベントを防止。  
  * ※印刷モード時のみ、これらの制限をCSSで解除 (user-select: text)。

## **4\. 技術仕様**

### **4.1. 外部ライブラリ (CDN)**

* **Tailwind CSS (v3.x):** スタイリングおよびダークモード制御。  
* **Chart.js:** グラフ描画。  
* **Plotly.js:** (読み込みのみ、現在の実装ではChart.jsがメインで使用されている)。  
* **FontAwesome:** アイコンフォント。  
* **Google Fonts:** "Noto Sans JP" (日本語フォント)。

### **4.2. ブラウザ互換性**

* Chrome, Edge, Safari, Firefox の最新版での動作を想定。  
* IEはサポート対象外。

### **4.3. 運用・保守**

* 単一のHTMLファイル (index.html) を編集するだけで内容の更新が可能。  
* 画像アセット等は使用せず、CSS装飾とフォントアイコンのみで構成されているため、リンク切れのリスクがない（QRコード生成APIを除く）。
