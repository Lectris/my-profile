# **ポートフォリオ & 職務経歴書**

Webポートフォリオサイトです。

Web上での閲覧に適した「没入型ポートフォリオ」と、採用担当者への提出に適した「A4印刷用職務経歴書」の2つの顔を持つ、シングルページアプリケーション（SPA風）として設計されています。

## **特徴**

* **ハイブリッドデザイン:**  
  * **Web View:** ダークモードを基調とした、アニメーションとインタラクティブなチャートを含むリッチなUI。  
  * **Print View:** ブラウザの印刷機能 (Ctrl+P) を使用すると、自動的に「JIS規格履歴書・職務経歴書」スタイルへレイアウトが変換されます。  
* **レスポンシブ対応:**  
  * PC: 排他的アコーディオン機能を備えた固定サイドバー。  
  * Mobile: スクロール追従型スティッキーヘッダーとボトムナビゲーション。  
* **テーマ切り替え:** ヘッダーのトグルボタンにより、ライトモード/ダークモードの切り替えが可能（設定はブラウザに保存）。  
* **セキュリティ対策:** 検索エンジン除外 (noindex) および、簡易的なコピーガード（右クリック・選択禁止）を実装。

## **技術スタック**

ビルド不要で動作するよう、CDNベースの構成を採用しています。

* **Core:** HTML5, Vanilla JavaScript  
* **Styling:** Tailwind CSS (CDN)  
* **Visualization:** Chart.js, Plotly.js  
* **Icons:** FontAwesome  
* **Font:** Google Fonts (Noto Sans JP)

## **使用方法**

1. index.html をモダンブラウザ（Chrome, Edge, Safari等）で開いてください。  
2. **Web閲覧:** スクロールして経歴やスキルを確認できます。右上のアイコンでテーマを変更できます。  
3. **印刷/PDF保存:** Ctrl \+ P (Macは Cmd \+ P) を押し、プレビューを確認してください。不要な装飾が消え、提出用のフォーマットになります。

## **ディレクトリ構成**

本プロジェクトは index.html 1ファイルのみで完結しています。CSSおよびJavaScriptはHTML内にインラインで記述されています。
