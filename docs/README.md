# **Portfolio Project**

本リポジトリは、自身の職務経歴書およびポートフォリオを生成・公開するためのプロジェクトです。

「提出用のPDF資料」と「Web公開用のポートフォリオサイト」を、Web技術（HTML/CSS/JS）を用いて一元的に管理・作成しています。

## **📁 ディレクトリ構成**

.  
├── index.html      \# Web公開用ポートフォリオ（匿名化版・レスポンシブ）  
├── print.html      \# PDF出力専用ファイル（実名版・A4横固定レイアウト）  
├── README.md       \# 本ファイル  
└── SPECIFICATION.md \# 詳細仕様書

## **🚀 特徴**

### **📄 print.html (For PDF Export)**

採用担当者への提出を想定した、A4横スライド形式の職務経歴書です。

* **Pixel-Perfect Print:** ブラウザの印刷機能を使って、崩れのない完璧なレイアウトを出力します。  
* **Content Rich:** 詳細な職務内容、数値実績、保有資格を網羅しています。  
* **Privacy:** 実名・取引先名が含まれるため、\*\*非公開（ローカル運用）\*\*としています。

### **🌐 index.html (For Web)**

技術スキルを証明するための公開用Webサイトです。

* **Modern UI:** Bento Gridレイアウト、ダークモード、スクロールアニメーションを採用。  
* **Responsive:** PC、タブレット、スマートフォンすべてのデバイスに対応。  
* **Anonymized:** 個人情報を伏せた状態で、実績とスキルをアピールします。

## **📖 使い方**

### **1\. PDF資料の作成 (print.html)**

このファイルはブラウザで閲覧するためではなく、**PDFとして保存するため**に設計されています。

1. print.html を Google Chrome で開きます。  
2. 印刷ダイアログ (Ctrl/Cmd \+ P) を開きます。  
3. 以下の設定でPDF保存してください：  
   * **レイアウト:** 横 (Landscape)  
   * **用紙サイズ:** A4  
   * **倍率:** 既定 (100%) ※変更するとレイアウトが崩れます  
   * **背景のグラフィック:** **オン (必須)**  
   * **ヘッダーとフッター:** オフ

### **2\. Webサイトの公開 (index.html)**

GitHub Pages や Cloudflare Pages 等の静的ホスティングサービスにデプロイして使用します。

print.html が公開ディレクトリに含まれないように注意してください。
